BRAIN GAMES
===========

A PHP package with 5 games that make your brain work.

Installation  
------------
Via Composer

    $ composer global require valentina-vasileva/php-project-lvl1

Usage
=====

brain-even
----------
The goal of the game is indicating even numbers.

To play:  
    
    $ brain-even
